# 🛠️ Geliştirme Görevleri (Task List) - Micro Factory AI (DAN)

- [x] Proje klasör yapısı (index.html, style.css, script.js) ve Tailwind CSS bağlantısının kurulması.
- [x] Modern Glassmorphism UI ve Indigo-Violet "Premium SaaS" arayüz tasarımının uygulanması.
- [x] **B2B Sipariş Formu:** 81 il/ilçe destekli dinamik teslimat rotası seçimi ve 5 ana kategorinin (Doğal Kozmetik dahil) forma eklenmesi.

- [x] **Hub Veritabanı:** Şahıs isimlerinden arındırılmış, 7 bölgeye yayılmış "Bölgesel Kooperatif Hub" veritabanının (JSON) kurulması.

- [x] **Veri Kalıcılığı:** `localStorage` kullanılarak sipariş geçmişi, SLA puanları ve timeline verilerinin tarayıcıda tutulması.

### 🧠 Phase 3: AI Entegrasyonu (Gemini 2.5 Flash)
- [x] Yapay Zeka API (Gemini) bağlantısı ve Karar Motoru (System Prompt) mimarisinin inşası.
