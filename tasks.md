# 🛠️ Geliştirme Görevleri (Task List) - Micro Factory AI

Bu liste, projenin prototip aşamasından canlıya alınmasına kadar olan teknik adımları içerir.

### ✅ Phase 1: Temel Yapı ve Arayüz (UI/UX)
- [ ] 1.1. Proje klasör yapısının oluşturulması (index.html, style.css, script.js).
- [ ] 1.2. Tailwind CSS bağlantısının yapılması ve ana sayfa düzeninin kurulması.
- [ ] 1.3. "Kurumsal Sipariş Giriş Formu"nun (Ürün adı, Adet, Karmaşıklık) tasarlanması.
- [ ] 1.4. "Üretim Dağıtım Tablosu" için şık bir yer tutucu (placeholder) alan oluşturulması.

### ⚙️ Phase 2: Veri ve Mantık (Data & Logic)
- [ ] 2.1. 20 kişilik sentetik kadın üretici veritabanının (JSON) hazırlanması.
- [ ] 2.2. Formdan gelen sipariş verilerini yakalayan JavaScript fonksiyonunun yazılması.
- [ ] 2.3. Basit bir "Kapasite Kontrol" algoritması (Toplam kapasite vs. Sipariş) eklenmesi.

### 🧠 Phase 3: AI Entegrasyonu (Intelligence)
- [ ] 3.1. Yapay Zeka API bağlantısı için gerekli altyapının kurulması.
- [ ] 3.2. Üretim dağıtımını yapacak olan "System Prompt"un (Planlama Mantığı) yazılması.
- [ ] 3.3. AI'dan gelen JSON formatındaki dağıtım planının ekrandaki tabloya yansıtılması.

### 🚀 Phase 4: Final ve Yayınlama (Launch)
- [ ] 4.1. Tasarımın mobil cihazlara uygunluğunun (Responsive) kontrol edilmesi.
- [ ] 4.2. Uygulamanın Lovable veya Netlify üzerinden internete yayınlanması.
